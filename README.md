# BrowserForge AI

> Transform your ideas into deployable web applications using AI-powered agents

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node](https://img.shields.io/badge/node-%3E%3D20.0.0-green.svg)

BrowserForge AI is a full-stack web application that lets users describe an app in natural language and get a complete, deployable product generated by browser-empowered SuperAgents.

## Features

### AI-Powered Generation
- **Multi-Agent Pipeline**: Researcher, Coder, Tester, and Deployer agents work together
- **Smart Code Generation**: Creates complete Next.js applications with Tailwind CSS and shadcn/ui
- **Research Agent**: Uses Playwright to scrape real documentation and best practices

### Live Development
- **Real-time Editor**: Monaco Editor with syntax highlighting
- **Live Preview**: Embedded iframe preview of generated apps
- **Code Editing**: Modify generated code directly in the browser

### One-Click Deploy
- **Vercel Integration**: Auto-generates vercel.json and deploys
- **Netlify Support**: Alternative deployment to Netlify
- **Docker Support**: Containerized deployment ready

### Security & Safety
- **Docker Sandboxing**: All code execution in isolated containers
- **Resource Limits**: Configurable CPU/memory limits per forge
- **Timeout Protection**: Automatic container cleanup after timeout

### Project Management
- **History**: Save and manage all your forges
- **Public Gallery**: Share and fork community templates
- **Private Projects**: Your ideas stay yours

## Quick Start

### Prerequisites

- Node.js 20+
- Python 3.11+
- Docker & Docker Compose
- pnpm (recommended) or npm

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/browserforge-ai.git
cd browserforge-ai
```

2. **Install dependencies**
```bash
pnpm install
```

3. **Set up environment variables**
```bash
cp .env.example .env
# Edit .env with your API keys
```

4. **Start with Docker Compose**
```bash
docker-compose up -d
```

Or run services individually:

```bash
# Frontend
cd app && pnpm dev

# Backend
cd api && uvicorn main:app --reload
```

5. **Open the app**
- Frontend: http://localhost:3000
- API: http://localhost:8000
- API Docs: http://localhost:8000/docs

### Using the Editor

1. Create an account or sign in
2. Click "New Project" and describe your app
3. Click "Forge" to generate code
4. Watch the AI agents work
5. Edit the generated code in the editor
6. Click "Deploy" to go live

## Configuration

### Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `DATABASE_URL` | SQLite or PostgreSQL connection | `sqlite+aiosqlite:///./browserforge.db` |
| `REDIS_URL` | Redis connection URL | `redis://localhost:6379` |
| `JWT_SECRET` | JWT signing secret | (required) |
| `OPENAI_API_KEY` | OpenAI API key | (optional) |
| `ANTHROPIC_API_KEY` | Anthropic API key | (optional) |
| `VERCEL_TOKEN` | Vercel deployment token | (optional) |
| `NETLIFY_TOKEN` | Netlify deployment token | (optional) |

### Docker Resource Limits

| Variable | Description | Default |
|----------|-------------|---------|
| `SANDBOX_CPU_LIMIT` | CPU limit per sandbox | `0.5` |
| `SANDBOX_MEMORY_LIMIT` | Memory limit per sandbox | `512m` |
| `SANDBOX_TIMEOUT` | Timeout in seconds | `300` |

## Deployment

### Docker Compose (Self-hosted)

```bash
docker-compose -f docker-compose.yml up -d
```

### Vercel (Frontend)

```bash
cd app
vercel --prod
```

### Railway / Render (Full Stack)

1. Connect your GitHub repository
2. Set build command: `cd api && pip install -r requirements.txt`
3. Set start command: `cd api && uvicorn main:app --host 0.0.0.0 --port $PORT`
